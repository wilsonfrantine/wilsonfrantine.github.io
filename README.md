# TATAbox Hub 🧬💻

**TATAbox** é o laboratório digital e hub pessoal de recursos de **Wilson Frantine**. Este espaço nasceu do desejo de organizar e compartilhar minha produção técnica em **Genética, Ecologia e Bioinformática** de uma forma que seja útil tanto para o meu "eu do futuro" quanto para colegas e estudantes da área.

## 🔬 O que você encontra aqui?

Diferente de um portfólio estático, o TATAbox é uma "bancada de trabalho" (Workbench) viva, onde compartilho:

- **Minhas Trilhas de Aprendizado**: Organização lógica dos meus materiais didáticos e scripts (como o ENM101 e R101).
- **Bioinfo Hub**: Pequenas ferramentas e utilitários que desenvolvo para facilitar o dia a dia da análise genômica.
- **Data Ecology**: Scripts de análise e pacotes R voltados para ecologia quantitativa.
- **Caderno de Laboratório**: Uma seção de tutoriais e dicas rápidas (Snippets) sobre bioinformática e ciência de dados.

## 🛠️ Bastidores (Tech Stack)

Para manter este site rápido e eficiente (e para eu exercitar minhas habilidades de dev), utilizo:
- **Astro 6 + Tailwind 4**: Para um site ultra-veloz e com design técnico.
- **GitHub Actions**: O deploy é automatizado a cada atualização.
- **Ciência Aberta**: Todo o conteúdo e código aqui são abertos e voltados para a democratização do conhecimento científico.
