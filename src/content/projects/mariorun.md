## Criatividade Além do Terminal

Nem tudo no laboratório digital precisa ser análise de dados complexa. O **Mario Run** é um projeto "just for fun" que explora as capacidades de animação CSS e lógica de colisões em JavaScript puro, servindo como um excelente exercício de desenvolvimento front-end.

### Lógica e Interatividade

Construir um jogo, por mais simples que seja, exige um entendimento profundo do ciclo de vida da aplicação, detecção de eventos e manipulação do DOM. Este projeto demonstra como elementos visuais e interativos podem ser orquestrados para criar uma experiência de usuário divertida e fluida.

### Tecnologias Utilizadas

*   **HTML5/CSS3**: Estrutura e animações baseadas em *sprites*.
*   **JavaScript**: Lógica do jogo e controle de estado.
*   **GitHub Pages**: Disponível para jogar diretamente no navegador.

O Mario Run é a prova de que a programação também é uma ferramenta de expressão criativa e lazer.
