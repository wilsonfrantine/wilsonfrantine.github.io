## Ciência Cidadã: Aves de Cornélio Procópio

Preservar a fauna local começa pelo conhecimento. O projeto **Aves Urbanas** busca reunir e divulgar informações sobre as espécies de aves encontradas em Cornélio Procópio, Paraná, oferecendo um recurso educativo acessível para a comunidade e estudantes.

### Explorando a Biodiversidade Local

O projeto inclui um site interativo onde é possível visualizar características detalhadas das espécies locais, incluindo habitat, comportamentos marcantes e fotos para identificação.

### Funcionalidades do Projeto

*   **Identificação Visual**: Galeria de fotos e descrições para facilitar o reconhecimento em campo.
*   **Informação Científica**: Dados técnicos sobre nome científico e habitat.
*   **Interatividade**: Interface leve e intuitiva construída com tecnologias web modernas.

### Como Contribuir

A preservação da biodiversidade urbana é um esforço coletivo. Você pode participar enviando fotos de avistamentos na cidade ou informações adicionais sobre comportamentos específicos das aves locais para inclusão no site.

Este projeto é um exemplo de como a tecnologia pode ser uma ponte entre a pesquisa científica e a educação ambiental da comunidade.
