## Primeiros Passos em R

A linguagem R é o padrão de ouro para análise de dados biológicos. O projeto `R101` foi criado para servir como um guia amigável e focado em problemas reais, ajudando biólogos e ecólogos que desejam começar sua jornada na ciência de dados.

### O Desafio de Aprender Programação

Muitas vezes, iniciantes se perdem em cursos genéricos. O `R101` foca no que importa: manipular tabelas de espécies, realizar testes estatísticos básicos e visualizar resultados de campo.

### O que você vai dominar:

*   **Fundamentos**: Objetos, vetores, dataframes e indexação.
*   **Manipulação de Dados**: Introdução ao poder do `dplyr` e `tidyr`.
*   **Visualização**: Primeiros passos para criar gráficos elegantes com `ggplot2`.
*   **Estatística**: Testes de hipóteses básicos (t-test, ANOVA) aplicados a dados biológicos.

### Estrutura do Material

Este repositório contém uma série de scripts comentados que funcionam como uma aula interativa. Você pode rodar as linhas e ver os resultados imediatamente no RStudio.

O `R101` é o seu primeiro passo para se tornar um pesquisador independente e capaz de lidar com a complexidade dos dados modernos.
